# Latihan CSS & HTML Semantik

Repositori ini berisi berbagai latihan HTML dan CSS untuk memperdalam pemahaman struktur semantik, styling dasar, serta manipulasi elemen-elemen umum dalam web development. Latihan ini merupakan bagian dari pengembangan kemampuan dasar front-end.

### 2. **blockquote.html**
Latihan penggunaan elemen:
- `<blockquote>` untuk kutipan
- `<q>` untuk kutipan pendek

- Styling kutipan menggunakan CSS agar tampak lebih profesional

### 3. **gambar.html**
Latihan yang memfokuskan pada:
- Penggunaan elemen `<img>`
- Atribut gambar seperti `alt`, `width`, dan `height`
- Penempatan gambar pada layout
- Styling gambar (border, shadow, positioning)

### 4. **huruf_kecil_bkebesar.html**
Latihan manipulasi teks, meliputi:
- Mengubah **uppercase / lowercase** dengan CSS (`text-transform`)
- Styling paragraf dan heading
- Pengaturan spacing, line-height, dan font-family
